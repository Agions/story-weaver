## 📋 关键决策记录

1. **useTheme ≠ useToken**: 之前报告说 "useTheme 是 useToken 的 shim" 是错的。useTheme 管 theme mode (light/dark), useToken 管 design tokens。不可互相替换。

2. **P4 报告 257 hex → 实际 188 hex**: P4 报告把 antd token 值算进去了。

3. **character-templates.ts 不可 token 化**: 27 处 hex 是角色外观数据 (肤色/发色/瞳色), 是内容非 UI。

4. **AntDSelect 保留**: 4 个文件用 antd 兼容 API (onChange/options/placeholder), 通过 select.tsx 内置的 LegacySelect 组件支持。

5. **剩余 121 hex 全部功能性**: 品牌色、角色数据、字幕默认值、场景氛围色, 强行 token 化会降低可读性。
