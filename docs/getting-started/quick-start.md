# 3 步跑通

> **5 分钟创建你的第一个 AI 漫剧**。本文带你完成完整流程，每步都有可执行操作。

---

## 一、上传故事

### 1.1 准备文本

支持格式：

| 格式     | 扩展名  | 推荐        |
| -------- | ------- | ----------- |
| 纯文本   | `.txt`  | ⭐⭐⭐ 最佳 |
| Markdown | `.md`   | ⭐⭐        |
| Word     | `.docx` | ⭐⭐        |
| PDF      | `.pdf`  | ⭐          |

### 1.2 文本建议

- **短篇最佳**：3000-10000 字（对应 3-8 分钟漫剧）
- **主角 1-3 个**，场景 5-10 个
- **情节清晰、对话丰富**（对话多 = 配音多 = 漫剧更丰满）

**示例输入**：

```text
在一个未来城市，机器人与人类共同生活。
主角小明是一台拥有自我意识的家务机器人，
他渴望了解什么是"家"的意义。

第 1 章：觉醒
某天，小明在打扫房间时，偶然听到了窗外的孩童笑声。
他第一次产生了疑问：为什么人类会笑？
```

### 1.3 上传

打开 Story Weaver → **新建项目** → **Autonomous 模式** → 拖入文件 / 点击选择

---

## 二、选择 AI 模型

### 2.1 文本模型（必需）

| 模型         | 价格 | 推荐场景             |
| ------------ | ---- | -------------------- |
| 智谱 GLM-5   | 💰   | 中文剧本首选         |
| Claude 3.5   | 💰💰 | 高质量长文本         |
| MiniMax M2.5 | 💰   | 备选                 |
| Kimi K2.5    | 💰   | 长上下文（10 万字+） |

> 💡 系统会**自动降级**：首选不可用时自动切换到备选 Provider。

### 2.2 图像模型（必需）

| 模型         | 推荐   | 备注                  |
| ------------ | ------ | --------------------- |
| Seedream 5.0 | ⭐⭐⭐ | 动画/插画首选（字节） |
| Kling 1.6    | ⭐⭐   | 视频生成（快手）      |
| Vidu Q3      | ⭐     | 高一致性（生数）      |

### 2.3 TTS 模型

**默认使用免费 Edge TTS**，无需配置 Key。也可选 CosyVoice 2.0（阿里付费）。

---

## 三、一键生成

### 3.1 配置选项

| 选项    | 推荐       | 备选              |
| ------- | ---------- | ----------------- |
| 质量    | Balanced   | Fast / High       |
| 分辨率  | 1080p      | 720p / 4K         |
| 宽高比  | 16:9       | 9:16（竖屏）/ 1:1 |
| 风格    | Cinematic  | Anime / Realistic |
| AI 模型 | 默认降级链 | 自定义排序        |

### 3.2 点击开始

```
开始 → ⏳ 等待（15-30 分钟）→ 🎬 成片
```

### 3.3 进度展示

```
✅ 1/11 导入       [100%]  10:00:30
✅ 2/11 分析       [100%]  10:01:15
🔄 3/11 脚本       [ 65%]  进行中
⏳ 4/11 角色
⏳ 5/11 场景
⏳ 6/11 分镜
⏳ 7/11 渲染
⏳ 8/11 视频剪辑
⏳ 9/11 配音
⏳ 10/11 字幕
⏳ 11/11 导出
```

> 💡 流水线每步自动 Quality Gate 评分，不合格会 Self-Review 重试（最多 3 次）。

---

## 四、下载成片

完成后系统**自动通知**，你可以：

| 操作            | 说明                    |
| --------------- | ----------------------- |
| **▶ 预览**      | 应用内播放              |
| **💾 下载**     | 保存为 MP4 / WebM / MOV |
| **🔄 重新生成** | 用不同参数重跑          |
| **✏️ 编辑**     | 跳转到 Manual 模式精调  |
