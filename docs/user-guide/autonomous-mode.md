# Autonomous 模式（全自动）

> **零参与一键成片** —— 你只需提供原材料，AI 自主完成全部 10 步流水线。

---

## 一、核心特性

| 特性              | 说明                                  |
| ----------------- | ------------------------------------- |
| 🕐 **零人工干预** | 提供原材料后无需任何操作              |
| 🧠 **AI 自审**    | 每步自动 Quality Gate，不合格自动修复 |
| 💾 **断点续传**   | 支持中途暂停、刷新继续                |
| 🔄 **自动降级**   | 主模型不可用时自动切换备选            |
| 📊 **实时进度**   | Web UI 实时展示每步状态               |
| 💰 **成本预估**   | 运行前自动估算 API 成本               |

---

## 二、使用流程

### 步骤 1：新建项目

打开 Story Weaver → 点击「**新建项目**」→ 选择「**Autonomous 模式**」

### 步骤 2：选择输入文件

支持格式：

| 格式     | 扩展名  | 适用场景   |
| -------- | ------- | ---------- |
| 纯文本   | `.txt`  | 原始小说   |
| Word     | `.docx` | Word 文档  |
| Markdown | `.md`   | 编剧脚本   |
| PDF      | `.pdf`  | 已发布作品 |
| 直接粘贴 | —       | 短篇/片段  |

### 步骤 3：配置基础选项

| 选项         | 推荐       | 说明                              |
| ------------ | ---------- | --------------------------------- |
| **质量预设** | Balanced   | fast / balanced / high            |
| **分辨率**   | 1080p      | 720p / 1080p / 4k                 |
| **宽高比**   | 16:9       | 9:16（短视频）/ 16:9（横屏）/ 1:1 |
| **风格**     | Cinematic  | cinematic / anime / realistic     |
| **AI 模型**  | 默认降级链 | 智谱首选，Claude 备选             |

### 步骤 4：点击「开始」

AI 立即开始 10 步流水线。你可以在 UI 上看到：

```
✅ 1/11 导入     [100%]  2026-06-10 10:00:30
✅ 2/11 分析     [100%]  2026-06-10 10:01:15
🔄 3/11 脚本     [ 65%]  进行中…
⏳ 4/11 角色     [  0%]
⏳ 5/11 场景
⏳ 6/11 分镜
⏳ 7/11 渲染
⏳ 8/11 视频剪辑
⏳ 9/11 配音
⏳ 10/11 字幕
⏳ 11/11 导出
```

### 步骤 5：等待完成

- 完成后系统**自动弹通知**
- 在「项目列表」中查看成片预览
- 一键下载 MP4 到本地

## 三、断点续传

如果中途发生：

- ❌ 应用崩溃
- ❌ 断网
- ❌ 切换设备

下次启动时 Story Weaver 会**自动检测**未完成项目，提示「**继续上次任务**」。

也可以手动管理：

```
设置 → 断点续传 → 列出所有 checkpoint → 选择要恢复的项目
```

## 四、Self-Review Loop 自审循环

每步输出后自动审核，**不通过则自动修复**：

```
[Step 3 脚本输出] → QualityGate
                          │
                ┌─────────┴──────────┐
                │                    │
            评分 0.88              评分 0.65
                │                    │
                ▼                    ▼
            PASS              Self-Review 触发
                                     │
                            ┌────────┴────────┐
                            │                 │
                        重试 1           评分 0.75
                       (优化 prompt)         │
                                              ▼
                                          重试 2 ...

                                  3 次仍 < 0.80
                                       │
                                       ▼
                                  记录日志 + 继续
```

**触发条件**：

| 评分维度   | 阈值   | 修复策略                   |
| ---------- | ------ | -------------------------- |
| 角色一致性 | < 0.85 | 重新生成参考图             |
| 视觉质量   | < 0.80 | 优化提示词 + 切换 Provider |
| 脚本对齐   | < 0.90 | 重读原文，补充上下文       |

## 五、成本控制

Autonomous 模式运行前会**预估成本**：

```
预估成本：¥8 - ¥15
  - 文本生成：¥1.5  (5 次调用)
  - 图像生成：¥4.0  (12 张参考图)
  - 视频渲染：¥6.0  (10 镜头 × 5s)
  - TTS 配音：¥0.5

  是否继续？[确认] [取消] [调整 Provider]
```

> 详见 [配置文档 - 降级策略](../getting-started/configuration.md#六降级链策略)

## 六、适用 vs 不适用

### ✅ 适用场景

- 短/中篇小说 → 漫剧快速成片
- 批量生产（多部短剧）
- 概念验证 / Demo
- 创作者快速预览效果
- AI 自审 + 断点续传，跑 1 小时不用盯

### ❌ 不适用场景

- 高度定制化分镜 → 改用 [Manual 模式](./manual-mode.md)
- 实时调整角色外观 → 改用 Manual
- 商业项目需要逐镜头调整 → 改用 Manual
- 客户中途要改剧本 → 改用 Manual
