## 📖 用户指南

### 入门指南
1. 安装依赖：`npm install`
2. 运行开发模式：`npm run dev`
3. 构建桌面应用：`npm run tauri build`

### 工作流程步骤
1. **导入** - 上传小说/脚本/提示词
2. **生成** - AI 脚本生成
3. **分镜** - 智能面板设计
4. **角色** - 角色设计
5. **渲染** - 场景渲染
6. **合成** - 动态合成
7. **导出** - 视频导出
