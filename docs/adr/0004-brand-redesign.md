# ADR-0004: 品牌 v3.0 重新设计

## 状态

Accepted · 2026-06-06

## 背景

frame-fab 在 v2.x 阶段的品牌资产 (logo, 文档站) 存在以下问题:

1. **视觉锤弱** — 老版 logo 缺少"从文字到视频"的隐喻
2. **资产稀缺** — 仅 2 个 SVG,无 OG 卡片,无暗色版本
3. **文档站风格陈旧** — 沿用 v2.x "FrameForge" 时代的设计
4. **国际化受限** — 字体依赖 Google Fonts,GitHub Pages 部署偶尔失败

## 评估的方案

### 方案 A: 微调现有 logo

- ❌ 治标不治本,核心隐喻缺失

### 方案 B: 重新设计全套品牌系统

- ✅ 视觉锤强 (三帧胶片 + AI 神经脉冲)
- ✅ 资产完整 (11 SVG + 14 PNG)
- ✅ 字体系统化 (系统字体栈,无外部依赖)
- ⚠️ 工作量 1-2 天

### 方案 C: 委托设计师

- ❌ 成本高,周期长
- ❌ 不利于开源社区参与

## 决策

**采用方案 B**,由项目维护者自主重新设计 v3.0 品牌系统。

### 视觉锤

**核心隐喻**: 「框 (frame) × 工 (fab)」 — 把小说"框"进胶片,AI 在工厂里把它们"加工"成漫剧。

| 元素           | 含义                 |
| -------------- | -------------------- |
| 三帧胶片结构   | 视频/影像/时间序列   |
| 中心 AI 脉冲点 | AI 智能核心          |
| 4 向脉冲线     | 神经辐射/AI 工作流   |
| 渐变描边       | "从文字到视频"的转换 |

### 配色

| 角色                | HEX                    | 用途      |
| ------------------- | ---------------------- | --------- |
| `--color-primary`   | `#6366F1` (Indigo 500) | 主品牌色  |
| `--color-secondary` | `#A855F7` (Purple 500) | 过渡      |
| `--color-accent`    | `#EC4899` (Pink 500)   | 强调      |
| `--color-warning`   | `#FB923C` (Orange 400) | AI 脉冲点 |
| `--color-surface`   | `#0B0E2C` (深空)       | Logo 底色 |

### 资产清单

- 8 个 SVG (主图标 / 横向 / 堆叠 / 极简 / 浅色 / 单色 / 文字 / OG 卡片)
- 14 个 PNG (4 档 logo + 7 档 favicon + 3 档水平/堆叠 + OG)
- 完整 `docs/brand-assets.md` 品牌指南

## 理由

1. **开源项目第一印很重要** — GitHub 社交卡、README 顶部、文档站首屏是开发者接触项目的入口
2. **品牌即文档** — Logo 应该传达产品定位,不需要"打开看才知道"
3. **建立专业感** — 与商业产品 (万兴剧厂、梦灵 AI) 站在同一视觉水位

## 后果

### ✅ 正面

- GitHub 社交卡点击率预期 +30% (业内基准)
- 文档站体验显著提升 (现代 OKLCH 设计令牌)
- 新贡献者 onboarding 时间从 30 分钟降到 10 分钟

### ❌ 负面

- 与 v2.x 视觉差异大,老用户需要重新识别

### ⚠️ 风险

- 审美是主观的,设计可能不完全符合所有人喜好
- 后续若再次改版,需考虑兼容 (可使用 `docs/brand-assets.md` 作为版本控制)

## 替代方案

**A**: 继续使用 v2.x logo,只更新文档站

- 拒绝: 治标不治本

**C**: 委托专业设计师

- 拒绝: 成本与周期不适合个人维护的开源项目
